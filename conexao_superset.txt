## 🔄 JÁ TEM A DATABASE? Como atualizar:

### Atualizar Conexão:
1. Settings → Database Connections
2. Clique no ícone de lápis (Edit) ✏️ na sua database PostgreSQL
3. Atualize as configurações
4. TEST CONNECTION → SAVE

### Atualizar Dataset (sincronizar colunas):
1. Data → Datasets
2. Encontre: superset_poligonos_covid
3. Três pontos (⋮) → Edit
4. Aba "Columns" → SYNC COLUMNS FROM SOURCE
5. SAVE

### Forçar Refresh no Chart:
- Abra o chart
- Três pontos (⋮) → Force refresh 🔄
- Ou pressione Ctrl + R
